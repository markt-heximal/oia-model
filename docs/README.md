# /docs — Supporting documentation

Reference and governance documentation for the model and the repository.

## Contents

- **`GLOSSARY.md`** — every canonical term: the ten layers, seven spans, three properties, four threat
  surfaces, the body of work, frameworks, primitives, and people.
- **`DESIGN-SYSTEM.md`** — the OIA design system: palette, typography, diagram conventions, and toolchain.
- **`ACKNOWLEDGMENTS.md`** — the author, architectural foundation, pressure-testers, and contributing partners.
- **`adr/`** — the [Architecture Decision Records](adr/) (0001–0022): the *why* behind the model and the
  repository, in Michael Nygard format. Start with [`adr/README.md`](adr/README.md).
