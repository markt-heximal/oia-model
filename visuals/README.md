# /visuals — Architectural diagrams

The visual reference for the model. Current renders are PDF; SVG versions are still to be produced
(see `MISSING-FILES.md`).

## Contents

- **`OIA-Visual-Reference-v0.5.pdf`** / **`.docx`** — the diagram set with explanatory text.
- **`OIA-Stack-v0.5.pdf`** — the ten-layer stack diagram (Layer 0 substrate, Layers 1–9 the stack).
- **`OIA-Threat-Surfaces-v0.5.pdf`** — the Four Named Threat Surfaces diagram (Mimicry, Agent-in-the-
  middle, Agent spoofing, Model compromise) and their concentration in the operational stack.
- **`OIA-Lens-Overlay-v0.5.pdf`** — the Lens Overlay showing how audience lenses map onto the architecture.
- **`OIA-Architecture-with-Framework-References-v0.5.pdf`** — the architecture annotated with the external
  frameworks it composes with (OSI, NIST, MITRE, OWASP, CCC, ISO/IEC 42001).

## The three canonical diagrams

Per the Body of Work Framing, the Visual Reference comprises three diagrams: **the Stack**, **the Lens
Overlay**, and **the Four Named Threat Surfaces**. These should be rebuilt as SVG for crisp web rendering.
