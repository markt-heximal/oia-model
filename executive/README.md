# /executive — Executive-facing materials

Materials engineered to make the OIA directly engageable by executives and boards without diluting the
architecture. See [ADR-0017](../docs/adr/0017-executive-accessibility-layer.md).

## Contents

- **`OIA-Overview-v0.5.pdf`** / **`.docx`** — the concise overview; the fastest accurate read of the model.
- **`OIA-Executive-Accessibility-Draft-v0.5.pdf`** — the v0.5.1 Executive Accessibility Enhancement:
  a one-page overview (readable in 60–90 seconds), a "why it matters" line per layer and span, and the
  **four named connections** (State-Holding Pair, Threat Concentration, Verifiability Chain, Three-Actor
  Identity Pattern).
- **`OIA-Thought-Leadership-Deck-v0.5.pdf`** / **`.pptx`** — the board/executive deck, calibrated for a
  30–45 minute session and **co-brandable** (a supporting-company slot on every slide footer). Three parts:
  the Reference Model, the Lens System, "What This Means for You."

## Reading order

1. `OIA-Overview-v0.5.pdf` (or the one-page overview inside the Executive Accessibility Draft).
2. `OIA-Executive-Accessibility-Draft-v0.5.pdf` — the four connections make the model memorable.
3. `OIA-Thought-Leadership-Deck-v0.5.pptx` — for presenting to a board or leadership audience.
